# დავალება 1:

დაამტკიცეთ, რომ ფუნქციის საშუალო ყოფელთვის მეტია ან ტოლია ფუნქციის მინიმუმის და ნაკლებია ან ტოლია ფუნქციის მაქსიმუმის:
minf <= Ef <= maxf

გვაქვს ფუნქცია f(a,b,c,d). რომელშიც ყოველთვის ხდება 3 შედარება, 1 დაბრუნება და მინიმუმ 1 მინიჭება(minf = 1).

ფსევდო კოდი:

largest = a ;
if ( largest < b )
largest = b ;
if ( largest < c )
largest = c ;
if ( largest < d )
largest = d ;
return largest ;

გამოვითვალოთ მინიჭებების საშუალო რაოდენობა, საშუალო არითმეტიკულის მეშვეობით.
მაქსიმუმ ხდება 4 მინიჭება(maxf = 4), შესაბამისად შედარებათა ჯამი გავყოთ მათ რაოდენობაზე:

(1 + 2 + 3 + 4) / 4 = 10 / 4 = 2.5

მივიღეთ 2.5 ანუ საშუალოდ მოხდება 2 ან 3 შედარება. ეს არ არის საშუალოს ზუსტი გამოთვლა, მაგრამ ჩვენ დავამტკიცეთ რომ:

minf <= Ef <= maxf
